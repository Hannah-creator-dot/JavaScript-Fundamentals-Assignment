# JavaScript Fundamentals Assignment

## 📌 Overview
This project demonstrates the core concepts of JavaScript, including variables, functions, loops, and DOM manipulation. It is organized into four parts as outlined in the assignment.

## 📂 Files
- `index.html`: The main HTML structure of the project.
- `style.css`: (Optional) Styling for the webpage.
- `script.js`: JavaScript logic covering:
  - Part 1: Variables, data types, conditionals
  - Part 2: Functions (2 examples)
  - Part 3: Loops (2 examples)
  - Part 4: DOM Manipulation (3 examples)

## ▶️ How to Run
1. Download and extract the zip file.
2. Open `index.html` in your browser.
3. Interact with the buttons and hover effects to see JavaScript in action.

## ✨ Features Demonstrated
- **Part 1 (Basics):** Variables, conditionals, and output.
- **Part 2 (Functions):** Reusable functions (`calculateTotal`, `greetUser`).
- **Part 3 (Loops):** Countdown loop and array iteration.
- **Part 4 (DOM Manipulation):** Toggle text, add list items dynamically, hover effect.
